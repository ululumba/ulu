План тестирования 


Цель:
Проверка базовой работоспособности учебного React-приложения.

Объект тестирования:
- Главная страница
- Интерактивные элементы (кнопки, формы)
- Отсутствие ошибок в консоли

 Метод:
Ручное тестирование в браузере Chrome.

 Среда:
- Локальный запуск через `npm run dev`
- Chrome последней версии

Отчёт по тестированию

Итог: 

1.Техническая стабильность на 100% — все пользователи смогли запустить приложение без ошибок
2.Интерфейс очень понятен — ни один респондент не отметил проблем с пониманием
3.Минимальные недочёты — у одного пользователя были мелкие замечания, что говорит о хорошем качестве
4.Высокая оценка и готовность к использованию — средний балл 4.3 и 83% готовых использовать — отличный результат

Рекомендации: 
Добавить короткое описание в интерфейс, чтобы ещё больше повысить ясность
Устранить мелкие недочёты, если они есть — например, добавьте анимацию или сообщение при действии
Попросить следующих пользователей указать, что именно было "мелким недочётом" — это поможет точнее улучшить UX


Проблемы и улучшения

Найденные проблемы:
- Нет заголовка или описания назначения
- Интерфейс выглядит как «голый шаблон»
- У некоторых — ошибки при загрузке (возможно, из-за развёртывания)

План улучшений:
- Добавить `<h1>My First React App — учебный проект</h1>`
- Вставить краткое пояснение: «Нажмите кнопку, чтобы увеличить счётчик»
- Добавить `<title>` в `index.html`
- Проверить, что все файлы попадают в Git


Тестирование и обратная связь

Проект протестирован вручную и протестирован пользователями (одногруппниками).

Что проверялось:
- Загрузка интерфейса
- Работа интерактивных элементов
- Удобство использования



Итог:
Приложение работает, но требует улучшения UX. Планируется добавить описание и визуальную обратную связь.
